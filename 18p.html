Parallel Circuits: Potential Difference, Current & Resistance

Notes (rewritten in English from the speech)

What is a parallel circuit?
  • A parallel circuit has more than one loop (more than one path for current).
  • In a truly parallel circuit, each loop usually contains one component.
  • Many real circuits are a mix of series and parallel, so some loops can contain more than one component.
More than one loop Multiple paths Often mixed circuits
Why parallel circuits are useful
  • Each component has its own loop, so if one component breaks, other loops can still work.
  • Example: if one light bulb in a house blows, the other lights stay on.
Potential difference (voltage) in parallel
  • In a truly parallel circuit, every component gets the full source potential difference.
  • If the supply is 12 V, then each component has 12 V across it.
Current in parallel
  • The total current is shared between the loops.
  • The currents in each loop always add up to the total: Itotal = I₁ + I₂ + ...
  • How the current splits depends on resistance: the loop with higher resistance takes a smaller share of the current.
Water analogy (least resistance)
  • Think of current like water flowing downhill through different paths.
  • More water (current) chooses the path of least resistance, so lower resistance loops carry higher current.
Total resistance in parallel
  • Adding more components in parallel makes the total resistance decrease.
  • Even if a new branch has a large resistance, it still gives the current an extra route, so overall it becomes easier for charge to flow.
  • So: more parallel loops → lower overall resistance.
