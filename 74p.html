Key Notes (rewritten from the speech)

1) What is ultrasound?
  • Ultrasound is sound with frequencies above 20,000 Hz.
  • It is beyond the range of human hearing.
2) Producing ultrasound
  • Humans produce ultrasound using electrical devices.
  • Electrical oscillations are converted into sound waves.
3) Reflection and transmission
  • When ultrasound hits a boundary, some waves are reflected and some are transmitted.
  • This is called partial reflection.
4) Using echoes
  • Reflected waves (echoes) can be used to calculate distance.
  • Distance is found using speed and time.
5) Medical use
  • Ultrasound is used in prenatal scanning.
  • It creates images of a fetus using reflected waves.
  • It is safe because it does not use radiation.
6) Industrial use
  • Ultrasound is used to detect cracks or faults in materials.
  • Unexpected reflections indicate defects.
7) Sonar
  • Used by ships and submarines to measure distance underwater.
  • Uses echo timing to calculate distance.
8) Distance calculation
  • Distance = speed × time.
  • For echoes, divide by 2 because the wave travels there and back.
